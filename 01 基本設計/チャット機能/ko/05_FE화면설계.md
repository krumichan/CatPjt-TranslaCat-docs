# FE 화면 설계

## 1. 화면 목록

| 경로 | 화면 | 설명 |
|---|---|---|
| `/chat` | 채팅방 목록 | 내가 참여 중인 채팅방 목록 |
| `/chat/[roomId]` | 채팅방 상세 | 메시지 송수신 화면 |

## 3. 채팅방 목록 화면

### 표시 항목

- 채팅방 이름
- 마지막 메시지
- 마지막 메시지 시간
- 참여자 수
- 미읽음 수. 2차 개발 대상

## 4. 채팅방 상세 화면

### 표시 항목

- 헤더
- 메시지 목록
- 번역문
- 메시지 입력창
- 언어 설정 버튼

## 5. 메시지 표시 규칙

### 내가 쓴 메시지

- 원문 표시
- 학습 언어 번역 표시

### 상대가 쓴 메시지

- 원문 표시
- 내 수신 번역 언어의 번역 표시

### 번역 대기 중

- `번역 중...` 표시 또는 작은 loading 표시

### 번역 실패

- `번역 실패` 표시
- 원문은 계속 표시

## 6. 메시지 로딩

- 채팅방 진입 시 최신 100개 메시지를 조회한다.
- 화면 표시는 오래된 메시지에서 최신 메시지 순으로 표시한다.
- 사용자가 위로 스크롤하면 과거 메시지 100개를 추가 조회한다.
- 추가 조회 후 현재 스크롤 위치가 튀지 않도록 보정한다.

## 7. WebSocket 처리

- 채팅방 진입 시 WebSocket에 연결한다.
- 현재 roomId topic을 subscribe한다.
- 페이지 이탈 시 unsubscribe 또는 연결 정리를 수행한다.
- 끊김 발생 시 재연결 UI를 표시한다.
