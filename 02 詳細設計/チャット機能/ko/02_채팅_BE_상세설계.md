# 채팅 BE 상세설계

## 1. 목적

본 문서는 채팅 기능 Backend 구현 기준을 정리한다.

## 2. 설계 범위

- 채팅방 관리
- 채팅방 멤버 관리
- 메시지 저장/조회
- WebSocket 송수신
- 비동기 번역 처리 연동

## 3. 패키지 구조 후보

```text
jp.co.translacat.domain.chat
```

## 4. 계층 구조

- Controller
- Facade
- CommandService
- QueryService
- DomainService
- Repository
- QueryRepository
- EventPublisher
- ExternalClient

## 5. 상세 내용

상세 내용은 BE 구현 단계에서 보강한다.
