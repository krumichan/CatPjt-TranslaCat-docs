# 채팅 가계부 연동 확장설계

## 1. 목적

본 문서는 향후 채팅방과 가계부를 연동하기 위한 확장 설계 방향을 정리한다.

## 2. 기본 방향

1차 개발에서는 가계부 연동을 구현하지 않는다.

단, 향후 그룹 채팅방 또는 오픈 채팅방에 하나 이상의 가계부를 연결할 수 있도록 채팅 도메인과 가계부 도메인을 강하게 결합하지 않는다.

## 3. 연결 테이블 후보

```text
chat_room_account_book_link
```

## 4. 설계 원칙

- chat_room에 accountBookId를 직접 두지 않는다.
- 채팅방 권한과 가계부 권한은 분리한다.
- 가계부 접근 권한은 account_book_member 기준으로 판단한다.

## 5. 상세 내용

3차 이후 개발 시 보강한다.
