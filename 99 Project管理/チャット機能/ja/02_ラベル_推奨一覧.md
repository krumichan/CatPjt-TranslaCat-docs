# GitHub Label 推奨一覧

各repoに以下のlabelを作成する。

## Phase

- `phase:1`
- `phase:2`

## Area

- `area:be`
- `area:fe`
- `area:docs`
- `area:qa`
- `area:infra`
- `area:ai`
- `area:common`

## Type

- `type:feature`
- `type:bug`
- `type:refactor`
- `type:docs`
- `type:test`
- `type:research`

## Priority

- `priority:high`
- `priority:medium`
- `priority:low`

## Status

- `status:blocked`
- `status:needs-review`
