# チャット機能 用語定義

| 用語 | 説明 |
|---|---|
| チャットルーム | 1:1、グループ、オープンチャットを含む会話空間 |
| 1:1チャット | 2名のユーザーが参加するチャットルーム |
| グループチャット | 複数ユーザーが参加する非公開チャットルーム |
| オープンチャット | 公開一覧からユーザーが参加できるチャットルーム。2次開発対象 |
| メッセージ原文 | ユーザーが実際に入力したメッセージ |
| 原文言語 | メッセージ原文の言語コード |
| 学習言語 | 自分が作成したメッセージ下部に学習目的で表示する翻訳言語 |
| 受信翻訳言語 | 相手のメッセージを自分が読みたい翻訳言語 |
| 翻訳PENDING | 翻訳要求は作成されたが、まだ完了していない状態 |
| 翻訳COMPLETED | 翻訳が正常完了した状態 |
| 翻訳FAILED | 翻訳が失敗した状態 |
| cursor pagination | offsetではなく、最後に取得したデータを基準に次のデータを取得する方式 |
| WebSocket | リアルタイムメッセージ送受信のための双方向通信方式 |
| AIチャットメンバー | チャットルームに参加するAI会話相手。2次開発対象 |
