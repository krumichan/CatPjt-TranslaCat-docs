# 채팅 기능 용어 정의

| 용어 | 설명 |
|---|---|
| 채팅방 | 1:1, 그룹, 오픈 채팅을 포함하는 대화 공간 |
| 1:1 채팅 | 두 명의 사용자가 참여하는 채팅방 |
| 그룹 채팅 | 여러 사용자가 참여하는 비공개 채팅방 |
| 오픈 채팅 | 공개 목록을 통해 사용자가 참여할 수 있는 채팅방. 2차 개발 대상 |
| 메시지 원문 | 사용자가 실제로 입력한 메시지 |
| 원문 언어 | 메시지 원문의 언어 코드 |
| 학습 언어 | 내가 작성한 메시지 하단에 학습 목적으로 표시할 번역 언어 |
| 수신 번역 언어 | 상대방 메시지를 내가 읽고 싶은 번역 언어 |
| 번역 PENDING | 번역 요청은 생성되었지만 아직 완료되지 않은 상태 |
| 번역 COMPLETED | 번역이 정상 완료된 상태 |
| 번역 FAILED | 번역이 실패한 상태 |
| cursor pagination | offset 대신 마지막으로 조회한 데이터 기준으로 다음 데이터를 가져오는 방식 |
| WebSocket | 실시간 메시지 송수신을 위한 양방향 통신 방식 |
| AI 채팅 멤버 | 채팅방에 참여하는 AI 대화 상대. 2차 개발 대상 |
